Use this project as a quick start for your Backbone.js apps.
*Bootstrap makes a guest appearance as the UI framework

Here's a full list of the libaries used in this project...

  • jQuery (v1.11.0)
  • Require.js (v2.1.11) - for JS dependency management
  • Backbone.js (v1.1.2) & Underscore.js (v1.6)
  • RequireJS text plugin (2.0.10) for loading templates
  • Moment.js for time functions
  • HTML5 shiv for Internet Explorer (v3.7.1)
  • Twitter Boostrap (v3.1.1)
  • Font awesome for scalable glyphs (v4.0.3)
